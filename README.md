## 🌋 What It Does

A humorous take on rental platforms set in a post-apocalyptic world. Browse listings for bunkers, fallout shelters, and fortified properties. Built as a creative full-stack project showcasing Next.js, dynamic routing, and responsive design.

## ✨ Features

- 🏚️ Browse post-apocalyptic rental listings
- 🔍 Filter by shelter type, location rating, and zombie proximity
- 📸 Listing detail pages with survival ratings
- 📱 Fully responsive design

## 🚀 Getting Started

```bash
git clone https://github.com/forbiddenlink/apoc-bnb
cd apoc-bnb
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
